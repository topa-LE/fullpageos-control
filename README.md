# FullPageOS Setup für Raspberry Pi 2 bis Pi5

Willkommen im FullPageOS-Setup-Repo! Dieses Repository enthält alles, was du benötigst, um **FullPageOS** auf verschiedenen **Raspberry Pi**-Modellen (Pi2, Pi3, Pi4, Pi5) zu installieren und zu konfigurieren.

## Links:
- [Deutsch (Haupt-README)](docs/README_de.md)
- [English (README in English)](docs/README_en.md)

### Weitere Informationen:
- **FullPageOS** ist ein leichtgewichtiges Betriebssystem für Kiosk-Anwendungen.
- Es ermöglicht es dir, Chromium im Kiosk-Modus auf einem Raspberry Pi zu betreiben, ideal für digitale Anzeigen oder Kiosk-Lösungen.

## Setup für Pi2, Pi3, Pi4, Pi5
- Die detaillierten Anleitungen und Skripte für jedes Pi-Modell findest du in den entsprechenden Verzeichnissen (`hardware/pi2`, `hardware/pi3`, etc.).
